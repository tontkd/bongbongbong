WebMail overview

MailBee WebMail Pro PHP is a web based mail client built as a set of PHP scripts.

MailBee WebMail Pro PHP allows users to access their local or remote e-mail accounts directly from web browser.

Viewing email securely, easy message composing and sending with multi-attachment support, advanced SPAM filtering, handy contacts management, multiple POP3/IMAP4/SMTP servers access along with many other great features make WebMail Pro the perfect solution for every email-enabled web sites and applications.

MailBee WebMail Pro PHP is shipped as source code, what allows you to use it "as-is" or modify the scripts to suit your needs.


 Advanced features:

  • Both AJAX and classic HTML user interface modes
  • IMAP4 Protocol support
  • AJAX auto-completion of To/Cc/Bcc fields
  • Integration with AfterLogic XMail Server
  • Blocking external images in e-mails for security reasons
  • Integration with external web applications
  • Advanced Address Book
  • Automatic marking messages as Read/Unread
  • WYSIWYG Editor (creates HTML messages in Rich Text mode)
  • Personal Signatures
  • SPAM filtering options
  • Multiple POP3/IMAP4/SMTP accounts access for each user
  • Personal user settings
  • Multi-lingual interface (english built-in, can be easily translated to another languages)
  • International charsets support
  • Timezone offset specification
  • Folders: folders management, folders navigation, moving e-mails between folders
  • Storing e-mails in database or on file system (configurable)
  • Databases supported: MySQL Server or MS SQL Server
 Basic features:
  • Access to any POP3/IMAP4/SMTP account from any browser
  • Compose/Reply/Forward/Redirect messages
  • Multiple attachments capability
  • Secure viewing HTML-formatted email messages, ability to view plain text version
  • Viewing and downloading files embedded or attached to the message
  • Printing and Deleting messages
  • "Show full headers" mode for displaying RFC822 message headers
  • Administrative web interface
  • The variety of WebMail skins
Benefits:
  • Perfect online email viewer
  • Easily integrates with your web site
  • Full source code provided
  • Easy customizable interface style
